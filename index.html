1. Introduction to Statistics

Definitions, types, branches, data, variables

Statistics — branch of mathematics dealing with collection, analysis and interpretation of numerical data for effective decision making.

Two main branches:
• Descriptive Statistics — organises & presents data (charts, tables, averages)
• Inferential Statistics — uses sample data to predict/conclude about a population

Data types:
• Primary data — collected directly from the field (raw, unprocessed)
• Secondary data — collected by someone else (NBS, CBN, WHO etc.)

Variables:
• Qualitative (Descriptive) — non-numeric (colour, gender)
• Quantitative (Numeric) — numeric: weight, height, temperature
  → Discrete (whole numbers only) | Continuous (any value in range)

Statistics can be represented in 3 dimensions (3D)
Statistics can be divided into 2 categories (Descriptive & Inferential)
📌 Population = entire group | Sample = subset selected for study
📌 Inferential statistics → Parametric & Non-parametric

2. Charts & Data Presentation

Pie chart, bar chart, histogram, multiple bar chart

Types of charts: Pictogram, Pie chart, Bar chart, Line graph, Histogram

Pie Chart: Angle of each sector = (Value / Total) × 360°
Angle = (f / N) × 360°
Histogram: 1-dimensional diagram; rectangles whose area ∝ frequency; width = class interval; no gaps between bars.

Bar Chart (Simple): Bars separated by equal spaces; height ∝ frequency.

Multiple Bar Chart: Grouped bars for comparison (e.g. male vs female per month).
📌 Compound/Percentage bar chart stacks values within each bar to show composition.

3. Frequency Distribution

Sturge's rule, class width, relative frequency

Sturge's Rule — determines number of classes (k):
k = 1 + 3.322 × log₁₀(N)
Class Width (C):
C = Range / k  =  (Max – Min) / k
Relative Frequency:
Relative Frequency = f / N  (or f/N × 100%)
Cumulative Frequency (CF): running total of frequencies.

📌 Range = Largest value − Smallest value
📌 Always round class width UP to next whole number

4. Measures of Central Tendency

Mean, Median, Mode — ungrouped & grouped

Arithmetic Mean:
x̄ = Σfx / N  (grouped)  |  x̄ = Σx / n  (ungrouped)
Geometric Mean:
G = ⁿ√(x₁·x₂·...·xₙ)
Harmonic Mean:
H = n / Σ(1/xᵢ)
In general: Harmonic Mean ≤ Geometric Mean ≤ Arithmetic Mean

Median (Grouped):
Median = L + [(N/2 – CF) / f] × C
Mode (Grouped):
Mode = L + [(Δ₁) / (Δ₁ + Δ₂)] × C
where Δ₁ = modal freq – prev freq, Δ₂ = modal freq – next freq

📌 For ungrouped data: Median = middle value when sorted; Mode = most frequent value
📌 Bimodal = 2 modes; Trimodal = 3 modes

5. Quartiles, Deciles & Percentiles

Q1, Q3, D1, D3, P1, P3 formulas

Quartiles (grouped data):
Q₁ = L + [(N/4 – CF) / f] × C
Q₃ = L + [(3N/4 – CF) / f] × C
Deciles:
D₁ = L + [(N/10 – CF) / f] × C
D₃ = L + [(3N/10 – CF) / f] × C
Percentiles:
P₁ = L + [(N/100 – CF) / f] × C
P₃ = L + [(3N/100 – CF) / f] × C
📌 L = lower class boundary, CF = cumulative frequency BEFORE the class, f = class frequency, C = class width

6. Measures of Dispersion

Range, Variance, Standard Deviation, CV

Range: Max − Min

Variance (ungrouped):
σ² = Σ(x – x̄)² / n
Variance (grouped):
σ² = Σf(x – x̄)² / N
Standard Deviation:
σ = √(Variance)
Coefficient of Variation (CV):
CV = (σ / x̄) × 100%
📌 CV is used to compare variability between two datasets with different means
📌 Lower CV = more consistent/less variable data

7. Probability

Classical probability, addition & multiplication rules

Classical Probability:
P(E) = Number of favourable outcomes / Total outcomes
Properties: 0 ≤ P(E) ≤ 1  |  P(E) + P(E') = 1

Addition Rule:
P(A∪B) = P(A) + P(B) – P(A∩B)
For mutually exclusive: P(A∪B) = P(A) + P(B)

Multiplication Rule:
P(A∩B) = P(A) × P(B|A)
For independent events: P(A∩B) = P(A) × P(B)

Combinations:
ⁿCᵣ = n! / [r!(n–r)!]
📌 With replacement → independent events. Without replacement → dependent events.
📌 Two dice: sample space = 36 outcomes

8. Probability Theory (Advanced)

Binomial distribution, Probability theorem

Probability Theorem: When a die is tossed, P(each face) = 1/6.
When coin is tossed, P(Head) = P(Tail) = 1/2

Binomial Distribution: For n trials, P(x successes) =
P(X=x) = ⁿCₓ × pˣ × (1–p)ⁿ⁻ˣ
Mean of Binomial = np  |  Variance = np(1–p)

📌 A fair coin: P(Head) = P(Tail) = ½

9. Correlation

Pearson's r, Spearman's rank correlation

Pearson's Correlation Coefficient (r):
r = [nΣxy – (Σx)(Σy)] / √{[nΣx²–(Σx)²][nΣy²–(Σy)²]}
Spearman's Rank Correlation:
r = 1 – [6Σd²] / [n(n²–1)]
where d = difference in ranks

Interpretation:
• r = +1 → Perfect positive correlation
• r = –1 → Perfect negative correlation
• r = 0 → No correlation
• |r| close to 1 → Strong correlation
📌 Correlation does NOT imply causation

10. Regression

Regression line of Y on X, Normal equations

Regression line of Y on X:
Y = a + bX
Normal Equations (solve simultaneously):
ΣY = na + bΣX
ΣXY = aΣX + bΣX²
Once a and b are found, substitute any X value to predict Y.

📌 b = slope (rate of change), a = Y-intercept
📌 n = number of data pairs

SECTION E — Probability

Two dice tossed simultaneously. Find the probability the sum is: (a) > 8 (b) Prime (c) Odd (d) < 10 (e) > 7 and < 9.
Total sample space = 36
(a) Sum > 8: sums 9,10,11,12 → outcomes: 4+3+2+1 = 10 → P = 10/36 = 5/18
(b) Prime sums (2,3,5,7,11): count pairs → 1+2+4+6+2 = 15 → P = 15/36 = 5/12
(c) Odd sums (3,5,7,9,11): count pairs → 2+4+6+4+2 = 18 → P = 18/36 = 1/2
(d) Sum < 10: sums 2–9 → 36 – 10 = 26 outcomes → P = 26/36 = 13/18
(e) Sum > 7 and < 9 (sum = 8): (2,6),(3,5),(4,4),(5,3),(6,2) = 5 → P = 5/36
